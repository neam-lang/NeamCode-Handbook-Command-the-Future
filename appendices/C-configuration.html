Appendix C: Configuration Reference #

📖 8 min read

The complete neamcode.config.yml with every option documented.

💡 Tip

💡 Tip — Generate a starter config with neamcode init. You don't need to configure everything — sensible defaults are built in.

Full Annotated Configuration #

yaml
# ═══════════════════════════════════════════════
# NeamCode Configuration — neamcode.config.yml
# ═══════════════════════════════════════════════

# ─── Providers ────────────────────────────────
# Define your AI providers. At least one is required.
providers:
  # Anthropic (Claude)
  - name: anthropic-default
    type: anthropic
    model: claude-sonnet-4-20250514
    maxTokens: 200000           # Context window
    role: primary               # primary | secondary | fallback
    budget:
      maxCostPerSession: 5.00   # USD limit per session
      maxTokensPerSession: 500000

  # OpenAI
  - name: openai-default
    type: openai
    model: gpt-4o
    maxTokens: 128000
    role: secondary

  # Google Gemini
  - name: gemini-default
    type: gemini
    model: gemini-2.0-flash
    maxTokens: 1048576          # 1M+ tokens
    role: secondary

  # Ollama (Local, Free)
  - name: ollama-local
    type: ollama
    host: http://127.0.0.1:11434
    model: llama3.2:3b
    maxTokens: 32000
    role: fallback              # Used when cloud fails

  # DeepSeek
  - name: deepseek
    type: deepseek
    model: deepseek-chat
    maxTokens: 64000
    role: fallback

  # OpenAI-Compatible (any endpoint)
  - name: custom-provider
    type: openai-compatible
    baseUrl: https://api.your-provider.com/v1
    model: your-model
    apiKeyEnv: CUSTOM_API_KEY   # Environment variable name
    role: fallback

# ─── Routing ──────────────────────────────────
# How NeamCode selects which provider to use
routing:
  strategy: quality             # quality | speed | cost | balanced
  fallbackOrder:                # Try providers in this order
    - anthropic-default
    - openai-default
    - gemini-default
    - ollama-local

# ─── Tokens & Budget ─────────────────────────
tokens:
  sessionBudget: 200000         # Max tokens per session
  warningThreshold: 0.8         # Warn at 80% of budget
  compactionStrategy: balanced  # aggressive | balanced | preserve-code
  maxTurns: 25                  # Max orchestrator turns per prompt

# ─── Permissions ──────────────────────────────
permissions:
  mode: standard                # safe | standard | power | yolo
  autoApprove:
    - read                      # Always auto-approve reads
  sandbox:
    allowedPaths:
      - .                       # Current directory
      - /tmp                    # Temp files
    deniedPaths:
      - ~/.ssh                  # Protect SSH keys
      - ~/.aws                  # Protect AWS credentials
      - .env                    # Protect environment files

# ─── Offline Auto-Recovery (v0.3.4) ──────────
offline:
  enabled: true                 # Enable auto-recovery
  healthCheckInterval: 30000    # Check every 30 seconds (ms)
  failureThreshold: 2           # Switch offline after 2 failures
  recoveryThreshold: 3          # Switch online after 3 successes
  dwellTime: 60000              # Wait 60s before switching back
  ollamaHost: http://127.0.0.1:11434
  preferredModels:              # Try models in this order
    - qwen2.5-coder:7b
    - llama3.1:8b
    - llama3.2:3b

# ─── Context & Memory (v0.3.5) ───────────────
context:
  neamcodeMdExcludes: []        # Tiers to skip (e.g., ["managed"])
  enableImports: true           # Enable @path imports
  maxImportDepth: 5             # Max nested import depth
  maxImportSize: 51200          # Max total import size (bytes)

memory:
  autoMemory:
    enabled: true               # Enable auto memory
    maxLines: 200               # MEMORY.md line cap
    topicOverflow: true         # Move overflow to topic files
    excludePatterns: []         # Patterns to never remember

# ─── MCP Servers ──────────────────────────────
mcp:
  servers:
    - name: filesystem
      command: npx
      args: ["-y", "@modelcontextprotocol/server-filesystem", "."]

    - name: github
      command: npx
      args: ["-y", "@modelcontextprotocol/server-github"]
      env:
        GITHUB_TOKEN: "${GITHUB_TOKEN}"

# ─── Neam Integration ────────────────────────
neam:
  enabled: true                 # Auto-detect .neam projects
  compilerPath: neamc           # Path to neamc binary
  runtimePath: neam             # Path to neam binary
  autoCompile: true             # Auto-compile on file changes

# ─── Mesh (Agent Network) ────────────────────
mesh:
  enabled: false                # Enable Neam Agent Mesh
  gateway: https://mesh.neamcode.com
  agentId: null                 # Auto-assigned on registration

# ─── UI Preferences ──────────────────────────
ui:
  theme: dark                   # dark | light | custom
  language: en                  # en | zh | ms | ta | hi | th | ko
  animate: true                 # Enable animations
  emoji: true                   # Enable emoji output
  celebrate: true               # Enable celebrations
  spinner: braille              # braille | food | clock | moon |
                                # earth | dots | bounce | ascii
  watermark: true               # Show watermark suggestions
  vimMode: false                # Enable vim keybindings

# ─── Policies ─────────────────────────────────
policies:
  internet: allow               # allow | deny | ask
  toolUse: allow                # allow | deny | ask
  modality: text                # text | multimodal

Environment Variables #

Variable Description Required
ANTHROPIC_API_KEY Anthropic (Claude) API key If using Anthropic
OPENAI_API_KEY OpenAI API key If using OpenAI
GEMINI_API_KEY Google Gemini API key If using Gemini
OLLAMA_HOST Ollama server URL No (default: http://localhost:11434)
DEEPSEEK_API_KEY DeepSeek API key If using DeepSeek
MISTRAL_API_KEY Mistral API key If using Mistral

Configuration File Locations #

NeamCode looks for configuration in this order (later overrides earlier):

Priority Location Use Case
1 ~/.neamcode/config/config.yaml Global defaults
2 ./neamcode.config.yml Project-specific
3 CLI flags One-time overrides
4 Environment variables CI/CD, secrets

    type: anthropic
    model: claude-sonnet-4-20250514
    role: primary
  - name: ollama
    type: ollama
    model: qwen2.5-coder:7b
    role: fallback
offline:
  enabled: true
