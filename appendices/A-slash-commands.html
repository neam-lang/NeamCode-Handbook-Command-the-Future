Appendix A: All Slash Commands #

📖 10 min read

A complete reference of every slash command available in NeamCode v0.3.5.

💡 Tip

💡 Tip — Type /help inside NeamCode to see this list at any time.

Help & Information #

Command Description Example
/help List all available commands /help
/help <command> Show detailed help for a command /help rewind
/doctor Run system health checks /doctor
/version Show NeamCode version /version

Session Management #

Command Description Example
/sessions List saved sessions /sessions
`/resume <id\ name>` Resume a saved session /resume my-project
/rename <name> Name the current session /rename auth-refactor
/clear Clear the conversation /clear
/export <file> Export conversation to file /export chat.md
/copy Copy last response to clipboard /copy

Context & Memory #

Command Description Example
/context Show context window usage /context
/compact Manually trigger context compaction /compact
/memory list List all memory entries /memory list
/memory search <q> Search memories /memory search jwt
/memory clear Clear all memory entries /memory clear
/memory export Export memory to JSON /memory export
/memory auto Show auto memory status (v0.3.5) /memory auto
/memory auto show Display MEMORY.md contents (v0.3.5) /memory auto show
/memory auto clear Clear auto memory for project (v0.3.5) /memory auto clear
/memory auto topics List topic overflow files (v0.3.5) /memory auto topics
/memory auto migrate Migrate legacy auto.json (v0.3.5) /memory auto migrate

Modes #

Command Description Example
/mode Show current mode /mode
/mode <name> Switch to a mode /mode explore
/mode off Return to default mode /mode off

Available modes: explore, debug, test, analyze, deploy, ideate

Code & Git #

Command Description Example
/diff Show git changes /diff
/review Automated code review /review
/rewind List available checkpoints /rewind
/rewind <n> Revert to checkpoint #n /rewind 3
/rewind last Revert to the most recent checkpoint /rewind last

Provider & Model #

Command Description Example
/models List available models /models
/cost Show session costs /cost
/offline Switch to local Ollama (v0.3.4) /offline
/offline models List available Ollama models (v0.3.4) /offline models
/online Switch back to cloud (v0.3.4) /online
/status Show connectivity and stats (v0.3.4) /status

Skills & Agents #

Command Description Example
/skill list List all skills /skill list
/skill info <name> Show skill details /skill info file_edit
/agents List available agents /agents

MCP Servers #

Command Description Example
/mcp list List connected MCP servers /mcp list
/mcp start <name> Start an MCP server /mcp start github
/mcp tools <name> List tools from a server /mcp tools filesystem
/mcp test <name> Test MCP server connection /mcp test github

Neam Language #

Command Description Example
/neam info Show Neam project info /neam info
/neam compile <file> Compile a .neam file /neam compile main.neam
/neam run <file> Compile and run /neam run hello.neam
/neam build Build entire project /neam build
/neam files List .neam files /neam files
/neam repl Start interactive REPL /neam repl
/pkg list List installed packages /pkg list
/pkg search <name> Search package registry /pkg search http
/pkg install <name> Install a package /pkg install std.http
/eval <expr> Evaluate a Neam expression /eval 2 + 2
/eval run <dataset> Run evaluation suite /eval run accuracy
/debug-neam <file> Start DAP debugger /debug-neam main.neam

Background Tasks #

Command Description Example
/tasks List background tasks /tasks
/tasks cancel <id> Cancel a background task /tasks cancel 3

UI & Preferences #

Command Description Example
/theme List available themes /theme
/theme <name> Switch theme /theme dark
